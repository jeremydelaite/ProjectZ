# ProjectZ

## Développement d'un jeu vidéo 2D en vue isométrique : 

Un shooter de survie par manches à la Call of Duty Zombies, dans un village des Ardennes ravagé pendant l'hiver 1944-45.

Style cartoon sombre et sanglant, arsenal 100% français d'époque (MAS 1935A, MAS 38, FM 24/29...), zombies de la Wehrmacht et système de manches à difficulté infinie (manches 40-50 pour les meilleurs joueurs, améliorations d'armes type Pack-a-Punch). 

## Stack : 

Phaser 3, TypeScript, Vite — déploiement web visé. Organisation : repo GitHub jeremydelaite/ProjectZ avec kanban (GitHub Projects), (game-design, technique, gameplay, assets, ui, polish).
